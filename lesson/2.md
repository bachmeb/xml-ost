##### Make a new directory
    mkdir ~/git/xml-ost/workspace/two
    
##### Change to the new directory
    cd ~/git/xml-ost/workspace/two

##### Make a new XML document
    vim phonebook.xml
```xml
<?xml version="1.0"?>
<!DOCTYPE PhoneBook>
<PhoneBook>
    <Listing>
       <First>Alex</First>
       <Last>Chilton</Last>
       <Phone>1-800-123-4567</Phone>
    </Listing>
</PhoneBook>
```

##### Validate the syntax
    xmllint -noout phonebook.xml
